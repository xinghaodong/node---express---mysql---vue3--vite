## 功能

-   后端 express mysql2
-   中间件 jwt 权限
-   tree 形菜单的增删改
-   用户增删改
-   登录
-   前端 vite5 + vue3 + js 快速开发
-   vue-router4
-   element-plus
-   pinia
-   axios 请求、响应拦截,封装,双 token 策略
-   vite 配置 生产、测试、开发
-   分离的路由和菜单设置
-   可折叠侧边栏
-   多标签页： 支持鼠标右键点击 刷新、关闭功、可拖拽
-   模块化全局状态管理
-   数据持久化：实现用户登录、路由菜单
-   切换全局基础组件尺寸
